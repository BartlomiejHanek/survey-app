# survey-app

Aplikacja webowa do tworzenia, zarządzania i wypełniania ankiet.

## Struktura projektu (skrót)

- backend/ — Express + Mongoose
- frontend/ — React + Vite

## Wymagania

- Node.js 18+ i npm
- Dostęp do MongoDB

## Instalacja

- git clone https://github.com/BartlomiejHanek/survey-app
- cd survey-app

## Pliki konfiguracyjne (.env)

- backend/.env

  - MONGO_URI
  - PORT
  - JWT_SECRET

- frontend/.env
  - VITE_API

## Uruchamianie (lokalnie)

1. Backend

   - cd backend
   - npm install
   - npm run dev

2. Frontend
   - cd frontend
   - npm install
   - npm run dev
